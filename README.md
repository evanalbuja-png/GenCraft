# GenCraft

GenCraft is an evolutionary simulation game developed as a final project for Stanford Code in Place.

Players manage a population of organisms living in hostile environments.

Every generation:

- Organisms are evaluated
- The fittest survive
- Survivors reproduce
- Random environmental events may occur
- Players can perform limited genetic interventions

The objective is to survive 100 generations without extinction.

## Features

- Four biomes
- Genetic traits
- Natural selection
- Mutation system
- Random environmental events
- Interactive genetic engineering
- Real-time dashboard using Matplotlib
- Final evolution graphs
